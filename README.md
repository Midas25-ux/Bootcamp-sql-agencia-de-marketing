# Bootcamp SQL Agencia Marketing

## Descripción general

Este proyecto reúne los ejercicios de SQL del bootcamp para una agencia de marketing.
Cada semana está organizada como un mini-proyecto independiente con su propio `proyecto.sql` y su `README.md`.

El objetivo es practicar SQL paso a paso: desde consultas básicas hasta analítica, datos jerárquicos, funciones avanzadas y optimización.

## Qué encontrarás aquí

Cada semana tiene:

- `proyecto.sql`: el archivo con el esquema de las tablas y las consultas que debes completar.
- `README.md`: la explicación del ejercicio, los objetivos y los criterios de evaluación.

Las carpetas de semana son:

- `week-06-funciones_de_agregacion/`
- `week-07-null_y_constraints/`
- `week-08-proyecto_integrador/`
- `week-09-joins_inner_join_y_left_join/`
- `week-10-cross_join_y_self_join/`
- `week-11-subqueries/`
- `week-12-ctes_y_case_when/`
- `week-13-ctes_recursivas/`
- `week-14-window_functions_ranking/`
- `week-15-window_functions_navegacion_y_vistas/`
- `week-16-indices_y_funciones_integradas/`

## Cómo usar este repositorio

1. Abre la carpeta de la semana que quieras hacer.
2. Lee el `README.md` de esa semana para entender qué pide el ejercicio.
3. Abre `proyecto.sql` y busca los comentarios `TODO`.
4. Escribe tus consultas en `proyecto.sql`.
5. Prueba tus consultas con el motor SQL que prefieras.

## Resumen de contenido por semana

- `week-06-funciones_de_agregacion/`: aprende a usar `SUM()`, `AVG()`, `COUNT()`, `MAX()` y `MIN()`.
- `week-07-null_y_constraints/`: trabaja con `NULL`, `NOT NULL`, `UNIQUE` y otras restricciones.
- `week-08-proyecto_integrador/`: une varias técnicas en un solo proyecto.
- `week-09-joins_inner_join_y_left_join/`: combina tablas con joins básicos.
- `week-10-cross_join_y_self_join/`: usa `CROSS JOIN` y `SELF JOIN` para relaciones especiales.
- `week-11-subqueries/`: practica subconsultas dentro de consultas más grandes.
- `week-12-ctes_y_case_when/`: usa CTEs y `CASE WHEN` para reportes más limpios.
- `week-13-ctes_recursivas/`: recorre jerarquías con `WITH RECURSIVE`.
- `week-14-window_functions_ranking/`: crea rankings con funciones de ventana.
- `week-15-window_functions_navegacion_y_vistas/`: usa `LAG()`, `LEAD()` y crea vistas.
- `week-16-indices_y_funciones_integradas/`: aprende índices y funciones útiles de SQL.

## Consejos para trabajar

- Lee bien el README de cada semana antes de empezar.
- Resuelve los `TODO` dentro de `proyecto.sql`.
- Añade comentarios en español que expliquen lo que hace cada consulta.
- No uses `SELECT *` si debes mostrar columnas específicas.
- Prueba tus consultas con datos reales y suficientes.
