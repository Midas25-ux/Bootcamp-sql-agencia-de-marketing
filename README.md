# Bootcamp SQL - Agencia de Marketing

Este repositorio reúne los ejercicios y proyectos del bootcamp de SQL enfocados en el análisis de datos para una agencia de marketing.

## ¿Qué incluye?

El contenido está organizado por semanas y cada carpeta contiene:

- un archivo README con la explicación de la práctica
- un archivo SQL con los ejercicios o soluciones propuestos

## Estructura del proyecto

- week-06-funciones_de_agregacion
- week-07-null_y_constraints
- week-08-proyecto_integrador
- week-09-joins_inner_join_y_left_join
- week-10-cross_join_y_self_join
- week-11-subqueries
- week-12-ctes_y_case_when
- week-13-ctes_recursivas
- week-14-window_functions_ranking
- week-15-window_functions_navegacion_y_vistas
- week-16-indices_y_funciones_integradas

## Objetivo

Practicar conceptos clave de SQL como:

- funciones de agregación
- constraints y nulos
- joins
- subconsultas
- CTEs y CASE WHEN
- funciones ventana
- índices y funciones integradas

## Uso

1. Abre el archivo SQL correspondiente a la semana que quieras revisar.
2. Ejecútalo en tu motor de bases de datos preferido.
3. Usa el README de cada carpeta para entender el contexto y los objetivos del ejercicio.

## Ramas por semana

El proyecto también cuenta con ramas separadas por semana, de modo que cada rama conserva únicamente el contenido correspondiente a esa unidad.
